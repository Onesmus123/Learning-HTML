HTML Elements

An HTML element is everything from the start tag to the end tag.

Nested HTML Elements

A nested HTML Element is an element within another HTML Element.

Example

The HTML <title> element is usually nested inside the <head> element.

Empty Elements

These are HTML elements with no content.

Example

the <br> element

HTML is Not Case Sensitive

HTML tags are not case sensitive: <P> means the same as <p>.

NOTE: W3C recommends lowercase in HTML.
